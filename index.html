SVG - Scalable Vector Graphics


#be

SVG stands for Scalable Vector Graphics, and it is a markup language used to describe two-dimensional vector graphics.

Unlike raster images (JPG, PNG, etc.), SVG graphics are resolution-independent, meaning they can scale without loss of quality.

SVG elements are treated as DOM (Document Object Model) objects, allowing easy manipulation and interaction using JavaScript.

Canvas


Let's see what else we can do :P

Canvas is a powerful element in web development that allows for dynamic and interactive graphics rendering. Compared to other techniques like SVG, canvas is often preferred for animation due to its performance benefits

Canvas animations are more performant because they use a bitmap rendering approach. Instead of retaining a DOM structure like SVG, canvas directly manipulates pixels, resulting in faster rendering and smoother animations

Canvas provides more control over animations, allowing developers to create complex visuals and intricate motion effects. It's particularly useful for creating particle systems, fluid simulations, and other intricate animations
